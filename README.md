# codemirror-kage
codemirror with kage dynamic glyph

    cd /ksana2016
    git clone https://github.com/ksanaforge/codemirror-kage.git
    cd codemirror-kage
    npm i
    ./build-kage
    open nginx and listen to 2561
    
    npm start

    open http://127.0.0.1:2561/codemirror-kage/ via chrome
